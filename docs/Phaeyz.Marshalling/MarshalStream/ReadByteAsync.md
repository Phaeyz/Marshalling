# MarshalStream.ReadByteAsync method

Reads a byte from the current stream and advances the position within the stream by one byte.

```csharp
public ValueTask<int> ReadByteAsync(CancellationToken cancellationToken = default)
```

| parameter | description |
| --- | --- |
| cancellationToken | A cancellation token which may be used to cancel the operation. |

## Return Value

Returns a task which yields the byte which has been read in integer form, or `-1` if the end of the stream has been reached.
