# MarshalStream.AddWriteProcessor method

Registers a new write processor.

```csharp
public IDisposable AddWriteProcessor(IMarshalStreamProcessor processor)
```

| parameter | description |
| --- | --- |
| processor | The write processor to register. |

## Return Value

A IDisposable which may be used to remove the write processor.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | The write processor is already registered. |

## Remarks

When the write processor is registered, the stream will become unseekable and write-only. Multiple write processors may be concurrently registered.
