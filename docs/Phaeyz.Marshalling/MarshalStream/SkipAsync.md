# MarshalStream.SkipAsync method

Reads and discards a sequence of bytes from the current stream and advances the position within the stream by the specified number of bytes.

```csharp
public ValueTask<long> SkipAsync(long byteCount, CancellationToken cancellationToken = default)
```

| parameter | description |
| --- | --- |
| byteCount | A positive number indicating the number of bytes to skip. |
| cancellationToken | A cancellation token which may be used to cancel the operation. |

## Return Value

The number of bytes actually skipped, which may be less than the requested number of bytes if the end of the stream was reached.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentOutOfRangeException | *byteCount* is negative. |
| IOException | An I/O error occurred. |
| NotSupportedException | The stream is not readable. |
| ObjectDisposedException | The stream is disposed. |
